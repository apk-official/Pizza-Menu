## 🚀 How to clone this repo

1. Clone the repo
   ```sh
   git clone https://github.com/yourusername/pizza-menu.git
   ```
2. Navigate to the project directory
   ```sh
   cd pizza-menu
   ```
3. Install dependencies
   ```sh
   npm install
   ```
4. Run the project
   ```sh
   npm start
   ```
